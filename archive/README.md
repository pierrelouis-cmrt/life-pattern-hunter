# Archive

Ce dossier garde les fichiers utiles pour comprendre l'historique du projet, mais qui ne font plus partie du parcours principal.

- `tools/solution_cleaner.py` : outil conservateur pour nettoyer une grille initiale déjà trouvée.
- `tools/random_bruteforce_reference.py` : ancienne référence par force brute aléatoire, utile pour comparer avec le solveur génétique.
- `presentation_video/` : ancien support vidéo Manim et rendu MP4.
- `project_instructions.md` : anciennes consignes de projet.
- `pattern-hunter.py` et `tests/` : version historique avant la séparation claire entre algorithme, interface et contrôleur.
